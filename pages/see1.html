KlikShop is a simple yet modern e-commerce mobile application built using Flutter with GetX architecture. The app is designed to provide users with a seamless shopping experience, from browsing products to completing transactions.

Key Features

  • Authentication System Secure login and registration with Google integration.
  • Product Catalog Clean and minimal UI to browse and search products easily.
  • Product Detail Page Display product images, descriptions, and pricing with an intuitive layout.
  • Shopping Cart Add, manage, and review selected items before checkout.
  • Order History Track past transactions with clear status updates.
  • User Profile Manage account details and settings.

The UI/UX design follows a clean, minimalist, and modern style, inspired by Material Design with rounded corners, soft shadows, and consistent spacing. The app structure is modular, making it scalable and easy to maintain.

KlikShop demonstrates the integration of UI design (Figma) and clean Flutter codebase, showcasing both frontend skills and architectural understanding.

Technologies Used: Dart, Flutter, Firebase, GetX
